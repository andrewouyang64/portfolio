Works

Project 1 : Tourist-Info

This is an imformative tool for tourists to quickly obtain location and backgroud info for a city they may plan to visit. The application utilizes the Google map and Wikipedia by implementing API querying. It has a fully functional Google map with completely interactive while the major info of the city is extracted out from Wikipedia, as well as a link for further info.

The application provides user-friendly UI. It can record up to four previous searches.To re-visit a search just by clicking on the name on the list bar. New search will be added at the top with the first letter automatically converted to uppercase. A clear botton gives an option to clear out previous searches. By default, the city of Sydney is displayed when there is no previous search. When an invalid name is entered, an error message will popup.

The following technoloies have been implemented in this application: HTML, CSS, CSS framework from Tailwind, Javascript, jQuery and server side API data fetching. In this project, I was responsible for all Javascript structuring and rendering, as well as some CSS stylings.

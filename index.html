Works

Project 1 : Tourist-Info

This is an imformative tool for tourists to quickly obtain location and backgroud info for a city they may plan to visit. The application utilizes the Google map and Wikipedia by implementing API querying. It has a fully functional Google map with completely interactions while the major info of the city is extracted out from Wikipedia, as well as a link for further more info.

The application provides a user-friendly UI. It can record up to four previous searches.To re-visit a previous search just by clicking on the name on the list. After a new search, its name will be added at the top of the list. The first letter of the name is automatically converted to uppercase in display. A clear botton gives an option to clear out previous searches. By default, the city of Sydney is displayed. In case an invalid name is entered or a location is unavailable, a popup error message will alert you to try again.

The following technoloies have been implemented in this application: HTML, CSS, CSS framework from Tailwind, Javascript, jQuery and server side API data fetching. The design has responsive layout. I was responsible for all Javascript structuring and rendering, as well as some CSS stylings.
